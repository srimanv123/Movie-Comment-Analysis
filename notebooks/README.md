# Movie Comment Analysis

## Project Title
Survey-Based Analysis of Viewer Expectations, Genre Preferences, and Success Factors in Movies

## Description
This project analyzes movie comments using Machine Learning and Natural Language Processing techniques to predict movie ratings.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook

## Models Used

- Logistic Regression
- Balanced Logistic Regression
- Naive Bayes
- Random Forest

## Feature Extraction

- TF-IDF Vectorizer
